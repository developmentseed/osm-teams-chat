# OSM Teams Chat

A chat platform for OSM Teams.

## Development

Install requirements:

- [nvm](https://github.com/creationix/nvm)

Install Node.js the required version (see [.nvmrc](.nvmrc) file):

    nvm i

Install Node.js modules:

    yarn

Start the server:

    yarn dev
